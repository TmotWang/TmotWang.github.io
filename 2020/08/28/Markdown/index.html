流程flow

  1. 需要用flow语言进行编写```flow

  2. 格式 tag=>type: 内容:>url

    • tag为标签名

    • 元素种类type有6类

      • start
      • end
      • operation
      • subroutine
      • condition
        • 有yes和no两个分支
      • inputoutput
      • url为链接关联相应的框

  3. 用->来连接两个元素

  2. 需要指明为甘特图 gantt

  3. 日期格式 dateFormat YYYY-MM-DD

  4. 甘特图标题 title 名称

  5. 区域section 名称
